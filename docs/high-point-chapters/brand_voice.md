# High Point precedent — Brand Standards

> Source: HP_Franchise_Operations_Manual_v3_0.docx (Section 1 Brand Standards & Identity), HP_Location_BuildOut_Manual_v2.docx (Phase 04 Interior Design & Finishes, Phase 08 Brand Imaging Signage & Decor), HP_Employee_Handbook_v2.docx (Welcome + Hospitality Standard for tone-mirroring)

## What a strong Brand Standards chapter looks like

A precise, asset-driven chapter — typically 8–15 pages — that defines exactly what the brand looks like, sounds like, and stands for, and how franchisees are permitted to use brand assets. The strongest version covers four layers: (1) verbal identity (mission, vision, values, voice), (2) visual identity (logo, colors, typography, photography), (3) physical identity (interior design palette, materials, lighting, signage), and (4) usage rules (what franchisees may and may not do without approval). The job is to make brand consistency mechanical: any compliant franchisee can produce on-brand output by following the rules, with no aesthetic judgment required.

## Structure observed

- Brand story (one paragraph — origin, founding belief, what the name means)
- Mission / Vision / Core Values block (each a single line)
- Visual identity standards:
  - Logo usage rules (approved files only; never stretch / recolor / rotate / alter; clear-space rule defined; minimum size in print and digital; approved colorways enumerated)
  - Brand colors with full color-system spec (HEX / RGB / CMYK / Pantone) — primary and secondary
  - Typography (primary font, body font, optional script accent — single named font family with permitted weights)
- Brand voice standards: 4–5 named adjectives ("knowledgeable but not pretentious; warm but not over-familiar; energetic but not frantic; honest and direct")
- Prohibited brand uses: enumerated closed list (no co-branding without approval; no use of brand name in personal social media; no contests/sweepstakes/giveaways using the brand without approval; no menu modifications on branded materials; no signage outside approved templates)
- Physical / environmental identity (lives partly in the build-out manual but cross-referenced from the brand chapter):
  - Design philosophy as 3 named pillars (e.g. Craft / Warmth / Story) with one-sentence definitions
  - Approved color palette extended to environmental finishes (paint colors with manufacturer + product code)
  - Flooring, wall finish, ceiling, and lighting standards with named CCT temperature range (warm 2700–3000K maximum)
  - Pendant / sconce / track lighting requirements
  - Brand moments: feature wall / brand story wall / origin map / approved photography library / local artist mural permission process
- Signage standards: exterior (channel letter / illuminated cabinet / awning / blade sign / window graphics / door graphics / A-frame), interior (logo-bug placements, menu boards, wayfinding)
- Brand asset distribution: assets live in the franchisee Portal with metadata for approved use, expiration date, and channel restrictions
- Misuse / unauthorized use language tied to the Franchise Agreement: corrective action, including potential termination, for unauthorized brand use
- Approval workflow: any custom material requires submission to AOR or franchisor marketing team for review and final approval

## Depth indicators

- Color spec includes all four color systems (HEX / RGB / CMYK / Pantone) — supports print and digital.
- Logo clear-space rule is named geometrically ("equal to the height of the H in High") — survives any reproduction size.
- Minimum logo size is named in two units (inches in print / pixels in digital).
- Typography spec is one named font family per role — not "use a sans-serif."
- Brand voice is 4 adjectives, each with a contrast pair ("warm but not over-familiar") — gives a working filter for tone.
- Prohibited uses are a closed enumerated list — invites no interpretation.
- Environmental palette includes manufacturer-specific paint references (Benjamin Moore Aura or Sherwin-Williams Emerald grade) and named comparable paint products — survives the franchisee's local hardware store.
- Lighting CCT named in Kelvin range — testable with a meter.
- Pendant requirement: minimum 3 over standard bar length — a count, not "appropriately lit."
- Flooring permits named alternatives (engineered hardwood / matte porcelain / polished concrete) with technical thresholds (coefficient of friction 0.60 wet for BOH).
- Brand asset library is delivered through the Portal — versioned, expirable, channel-tagged.
- Approval workflow includes a named SLA (e.g. proof approval before any signage production; brand-team approval before any local mural commission).

## Tone & voice

Designer + lawyer hybrid. Specific and unambiguous on every visual asset; firm on usage rules; warm on the underlying values. Treats brand consistency as a mechanical output of following the rules, not a judgment call. The voice trusts the franchisee to execute correctly *if* given precise specs — so it provides them.

## What to avoid

- "Use the brand consistently" guidance without color codes, font names, or logo files — guarantees drift.
- Color specs in only one system (e.g. HEX only) — breaks across print and digital reproduction.
- Brand voice descriptions that are pure adjectives ("approachable, premium, modern") with no contrast pairs — every brand says these.
- Prohibited-use lists left as "use good judgment" — the absence creates the dispute.
- Aesthetic judgment calls embedded in the standards ("the design should feel warm") — replace with concrete specs (CCT range, finish materials, color palette).
- Brand assets distributed as a one-time PDF package — they need to live in a versioned Portal so updates can be pushed.
- Skipping the Franchise Agreement cross-reference — unauthorized brand use is a contractual matter, not just a marketing matter.
